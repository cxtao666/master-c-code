# C and C++ Practice

A personal collection of small C and C++ exercises created while learning core programming concepts.

The repository includes standalone programs covering input and output, control flow, functions, arrays, numeric operations, and Windows-oriented desktop experiments.

## Running an Example

Compile an individual source file with a C or C++ compiler:

```bash
gcc 123.c -o example
./example
```

Many files are independent exercises and may use different assumptions or toolchains.
